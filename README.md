# metamon_code
褒めたもんを実行するためのコード

## Requirement
tweepy==3.8.0  
dotenv==0.10.3

## How to USE

#### 1 clone this repository
~~~
git clone github.com/seven320/metamon_code
cd metamon_code
~~~
#### 2 get twitter api and input your api keys in .env like .env.exam
~~~
vim .env.exam
~~~
#### 3 run code
~~~
python hometamon.py
~~~
